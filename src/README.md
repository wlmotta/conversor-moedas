## 🌟 Funcionalidades do Projeto

🔄 Conversão entre moedas:

- [x] 🇺🇸 Dólar Americano (USD)
- [x] 🇦🇷 Peso Argentino (ARS)
- [x] 🇧🇷 Real Brasileiro (BRL)
- [x] 🇨🇴 Peso Colombiano (COP)

📡 Atualização automática via API (ExchangeRate-API)  
📴 Modo offline com taxas padrão caso a API não esteja disponível  
🖥️ Interface simples via linha de comando

## 🚀 Como Usar

- [x] ▶️ Execute o programa ConversorApp
- [x] 🔢 Selecione uma opção do menu:
****************************************************
1. 🇺🇸 Dólar → 🇦🇷 Peso argentino

2. 🇦🇷 Peso argentino → 🇺🇸 Dólar

3. 🇺🇸 Dólar → 🇧🇷 Real brasileiro

4. 🇧🇷 Real brasileiro → 🇺🇸 Dólar

5. 🇺🇸 Dólar → 🇨🇴 Peso colombiano

6. 🇴 Peso colombiano → 🇺🇸 Dólar

7. ❌ Sair
*****************************************************
- [x] 💵 Digite o valor que deseja converter
- [x] 🔄 O resultado será exibido automaticamente
- [x] ⏎  Pressione Enter para continuar ou 7 para sair

## ⚙️ Requisitos

- ☕ Java JDK 11+
- 🌐 Conexão com a internet (opcional - para taxas atualizadas)
- 📦 Dependência: Gson (para manipulação de JSON)

## 🔧 Configuração

🔑 Para usar sua própria chave de API:

1. Acesse ExchangeRate-API e obtenha uma chave
2. Substitua API_KEY na classe Conversor

## ⚠️ Limitações

- 📴 Offline: Usa taxas fixas (podem estar desatualizadas)
- 🌎 Moedas limitadas (mas pode ser facilmente estendido)
